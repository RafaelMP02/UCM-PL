\chapter{Tipos en C-Horizono}
\label{cap:tipos}
\section{Integer}
Corresponde con el \textit{signed int} de C++. Es decir, almacena números enteros representados en sistema decimal. Tienen un espacio reservado de 32 bits, 
por lo que su rango de valores es [-2147483648, 2147483647]. La palabra reservada para este tipo es  \Integer. Las operaciones que podremos realizar con 
enteros son: 
 

\begin{itemize}
	\item Suma. Suma natural de dos números enteros. Se utilizará el operador \Sum.
	\item Resta. Resta natural de dos números enteros. Se utilizará el operador \Sub. 
	\item Multiplicación. Multiplicación natural de dos números enteros. Se utilizará el operador \Mul.
	\item División. División entera de dos números enteros. Se truncará el resultado obtenido en caso de ser fraccionario. El operador utilizado será \Div.
\end{itemize}

\section{Boolean}
Corresponde con el \textit{bool} de C++. Almacenará valores verdaderos (\const{True}) o falsos (\const{False}). Se realizará conversión automática de tipo 
\Integer{} siendo "$0$" el entero que represente falso y cualquier otro número, verdadero. Las operaciones que podremos realizar serán:
 

\begin{itemize}
	\item Conjunción lógica. Su operador será el \And.
	\item Disyunción lógica. Su operador será el \Or.
	\item Negación lógica. Su operador será el \Not.
\end{itemize}
